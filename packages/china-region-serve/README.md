## 项目目录

```bash
  |__script // 脚本：bugatti
  |__src
  |    |__controller // 控制层
  |    |__db // 数据索引文件、数据缓存json文件
  |    |  |__**.json // 数据缓存文件
  |    |  |__region.sqlite // SQLite 数据库索引文档
  |    |  |__region.utf8 // jieba 词频字典
  |    |__filter // 拦截器
  |    |__interface // 实体定义
  |    |__provider // 服务提供者：dubbo\service\dto
  |    |__utils // 工具类
  |    |__main.ts // 项目入口
  |    |__module.ts // app.module
  |__CHANGELOG.md // change log
  |__CONTRIBUTION.md // 项目代码提交规范
  |__README.md // 项目说明文档
```
